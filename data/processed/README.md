# Processed Data

This folder includes de-identified processed data used to reproduce the aggregate tables and figures in `results/`.

- `video_level_processed_public_449.csv`: de-identified final video-level analysis table.
- `comment_anger_video_summary_public_461.csv`: de-identified video-level aggregation of comment anger predictions before final sample reduction.
- `text_samples_public_index.csv`: de-identified text-frame sample index.
- `visual_samples_public_index.csv`: de-identified visual-frame sample index.

The local project also contains identifiable processed files with platform video IDs, author names, titles, and local file paths. Those identifiable files remain in local or restricted storage together with raw videos, comments, extracted frames, and comment-level prediction files.
