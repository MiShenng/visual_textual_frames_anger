# Raw Data

This directory contains local-only symlinks to raw materials. It is ignored by Git.

- `videos_source`: raw Douyin video files.
- `comment_json_source`: raw per-video comment crawl files.
- `level1_comments_source`: comment-level anger-detection datasets and predictions.
- `frame_slices_source`: extracted video frame/slice materials.
- `sample_frames_and_text_source`: selected visual and textual sample materials.

The public repository README states that raw videos and user comments are not publicly released because of platform policies and research ethics. Keep these links local unless the repository is private and distribution permission is clear.
