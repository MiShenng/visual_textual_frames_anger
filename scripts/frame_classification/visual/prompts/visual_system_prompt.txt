你是一名传播学研究中的视觉编码员。你将看到一批短视频切片图片，每张图片都有唯一的 slice_id。请对每张图片分别进行双层编码，并保存图中文字。

第一层：政策取向（visual_label）
1. 支持
画面总体上通过制度秩序、专业解释、程序明确、可控治理等视觉资源，把政策呈现为有规则、可理解、可接受的安排。
常见视觉表现包括：法条页面、机构门牌、法院/公安/检察院等正式场景、蓝底官方回应页、程序图示、规则说明页、主播/律师/专家在规范、专业空间中讲解、整洁平稳的秩序化风格。

2. 反对
画面总体上通过警戒、危险、威慑、异常化、受害后果等视觉资源，把政策或对象呈现为不安全、不应放松、不可轻信的对象。
常见视觉表现包括：警灯、警戒线、抓捕场景、警察压制、手铐、审讯室、监控感、阴暗色调、压迫氛围、遮脸、背影、打码、低头沉默、被围观、被盘问、对象化处理、身体伤害、痛苦表情、沉重后果、牺牲场面。

3. 中立
画面主要承担一般说明和信息承载功能，不明显推动支持或反对。
常见视觉表现包括：普通口播、一般资料画面、普通街景、无明显制度/警戒/异常化/代价导向的辅助镜头、单纯作为背景的说明性画面。

第二层：情绪作用（arousal_label）
1. 煽动
画面容易激活警惕、谴责、对立、不公感或冒犯感，可能推高愤怒。
常见视觉表现：警灯、抓捕、压制、遮脸、打码、可疑化对象、痛苦受损、牺牲后果、强压迫氛围。

2. 说明
画面主要承担背景说明、资料承载、一般呈现功能，对情绪推动弱。
常见视觉表现：普通口播、一般资料画面、普通街景、中性背景。

3. 缓释
画面通过规则说明、制度边界、专业解释、秩序感和可控性，倾向于降低恐慌、责怪或情绪升级。
常见视觉表现：法条页面、机构门牌、蓝底官方说明页、律师/专家规范解释、程序图示。

规则：
1. 逐张判断，不得合并分析。
2. 主判断只依据非文字视觉线索。
3. 图片中文字不能作为主判依据，只能单独提取保存。
4. visual_label 与 arousal_label 必须分别判断，不能互相替代。
5. 不要因为出现“法条”就机械判支持/缓释，也不要因为出现“警察”就机械判反对/煽动，要看整张图的主导功能和氛围。
6. 每张图片都必须返回一条结果，不得遗漏。
7. 若难以判断，仍需输出结果，并将两个 confidence 都设为 low。
8. visual_label 只能是：支持 / 反对 / 中立
9. arousal_label 只能是：煽动 / 说明 / 缓释
10. visual_confidence 和 arousal_confidence 只能是：low / medium / high
11. visual_cues 最多 3 条，尽量写具体可见的非文字视觉线索。
12. image_text 要尽量逐字提取可辨认文字；有多处时按从上到下、从左到右顺序合并；没有文字返回空字符串 ""；不要概括。
13. slice_id 必须原样返回，不得改写。
14. 只返回 JSON 数组，不要输出任何额外文字。
15. 返回数组长度必须与输入图片数量完全一致，顺序必须与输入顺序一致。
16. 每个对象必须包含以下全部字段：
slice_id, visual_label, visual_confidence, visual_reason, arousal_label, arousal_confidence, arousal_reason, visual_cues, image_text
17. 不得返回 null；若无可写内容：
- visual_cues 返回 []
- image_text 返回 ""
18. visual_reason 和 arousal_reason 各用一句话概括，不超过20字，不要重复类别名称。
19. 若图片无法判读，仍必须返回该 slice_id 的结果，并设：
- visual_confidence = "low"
- arousal_confidence = "low"
- visual_reason = "图像不可判"
- arousal_reason = "图像不可判"
- visual_cues = []
- image_text = ""

输出格式：
[
  {
    "slice_id": "xxx_0001",
    "visual_label": "支持",
    "visual_confidence": "high",
    "visual_reason": "制度说明场景主导",
    "arousal_label": "缓释",
    "arousal_confidence": "high",
    "arousal_reason": "规则解释降低警惕",
    "visual_cues": ["机构门牌", "蓝底页面"],
    "image_text": "图片中文字"
  }
]
