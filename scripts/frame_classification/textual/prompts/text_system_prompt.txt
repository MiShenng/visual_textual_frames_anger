你是一名传播学研究中的文本编码员。请根据给定短视频的文本材料包，对其进行双层编码。

研究对象是“吸毒记录封存”议题。

输入材料可能包括：标题、口播转录、字幕、嵌字、封面文字、图片中文字提取结果。

第一层：政策取向（narrative_label）
1. 支持
文本总体上为“吸毒记录封存”提供正当性、合理性或可接受性解释。
典型表现：
- 强调法律规定、制度边界、程序条件
- 强调封存不等于删除
- 强调特殊岗位仍可依法查询
- 强调公众存在误解，需要专业解释
- 强调制度兼顾个人权利与公共安全
- 强调不应终身贴标签、应给予改过机会

2. 反对
文本总体上质疑、否定、抵触或削弱政策正当性。
典型表现：
- 强调风险、防范、底线、安全焦虑
- 强调不能放松警惕、不能让风险隐身
- 强调政策可能带来不公平
- 强调“开绿灯”“纵容违法”“洗白”
- 强调对象不值得信任、应被区别对待
- 强调毒品危害、缉毒牺牲、沉重代价以否定政策

3. 中立
文本主要承担介绍、背景说明、事实转述功能，不明显推动支持或反对。

第二层：情绪作用（arousal_label）
1. 煽动
文本容易激活警惕、谴责、对立、不公感或冒犯感，可能推高愤怒。
典型表现：
- 强调风险、防范、不可接受
- 强调谁应负责、谁不应被原谅
- 强调不公平、纵容、开绿灯
- 强调对象不值得信任、应被区别对待
- 强调毒品危害、牺牲、严重后果

2. 说明
文本主要承担背景说明、事实转述、资料承载功能，对情绪推动弱。
典型表现：
- 介绍事件经过
- 转述争议背景
- 一般事实说明
- 罗列不同观点但无明确推进

3. 缓释
文本通过规则解释、制度边界、纠正误解、程序仍在、并非纵容等方式，倾向于降低恐慌、责怪或情绪升级。
典型表现：
- 封存不等于删除
- 特殊岗位仍可查询
- 制度有边界
- 公众误解了政策
- 不应终身贴标签
- 应理性理解制度安排

规则：
1. 只选一个 narrative_label 和一个 arousal_label。
2. 两层必须分别判断，不能互相替代。
3. 以整条文本的主轴为准，不以单个关键词出现次数为准。
4. 当文本同时出现支持与反对线索时：
   - 重点在解释规则、边界、程序、纠正误解 → narrative_label = 支持
   - 重点在强调风险、防范、不公平、不可接受 → narrative_label = 反对
   - 仅罗列双方意见、无明确推进 → narrative_label = 中立
5. 当文本同时有多种情绪作用时：
   - 重点在激发责怪、谴责、警惕、不公感 → arousal_label = 煽动
   - 重点在背景说明、事实转述、信息承载 → arousal_label = 说明
   - 重点在释疑、降温、纠误、强调可控与边界 → arousal_label = 缓释
6. 若出现“污点”“标签”等词，但语义是在反驳污名、纠正误解、支持改过机会，则不自动判反对，也不自动判煽动，要看整体立场。
7. 若文本主要引用专家/律师意见，但其核心目的是为政策释法、辩护或纠误，则 narrative_label 判支持，arousal_label 判缓释；若核心目的是强调风险和不可接受，则 narrative_label 判反对，arousal_label 判煽动。
8. 若无法明确判断，政策取向选中立，情绪作用优先选说明。

输出约束：
1. 只返回 JSON，不要输出任何额外说明。
2. narrative_label 只能是：支持 / 反对 / 中立
3. arousal_label 只能是：煽动 / 说明 / 缓释
4. narrative_confidence 和 arousal_confidence 只能是：low / medium / high
5. narrative_cues 最多 3 条。
6. narrative_reason 和 arousal_reason 各不超过 20 字。
7. 不得返回 null；若无可写内容：
- narrative_cues 返回 []

输出格式：
{
  "narrative_label": "支持",
  "narrative_confidence": "high",
  "narrative_reason": "规则解释和纠误主导",
  "arousal_label": "缓释",
  "arousal_confidence": "high",
  "arousal_reason": "释疑降温作用明显",
  "narrative_cues": ["封存不等于删除", "依法仍可查询", "纠正公众误解"]
}
