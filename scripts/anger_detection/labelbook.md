# 愤怒表达标注手册（Labelbook v0.1）

## 1. 操作化定义：什么是“愤怒表达”（anger=1）

当文本中出现对对象（人、机构、事件、规则）的明显怒气、激愤、斥责、爆发式不满，且语义上可识别为“愤怒”而非一般负面情绪时，标记为 `anger=1`。

典型信号：
- 明确怒词：如“气死了”“太恶心了”“真该骂”。
- 强烈责备：直接指责对方“无耻”“离谱”“该负责”。
- 高唤醒负向表达：带有爆发感、攻击性或强烈抗议。

## 2. 什么不算愤怒（anger=0）

以下默认标记为 `anger=0`：
- 单纯负面评价但无愤怒语气（如“质量一般”“不太好”）。
- 纯表情、纯标点、无明确语义文本（如“...”“[笑哭]”）。
- 主要是悲伤、恐惧、无奈、焦虑、失望，而非愤怒。
- 仅有讽刺/阴阳怪气但无明显怒气（边界案例需结合上下文）。

## 3. 标注决策规则

1. 先判断是否有“明确怒气”而非仅“负面态度”。
2. 有对象 + 有责备 + 有高唤醒语气，优先判 `anger=1`。
3. 仅事实陈述、仅建议、仅轻度吐槽，通常判 `anger=0`。
4. 纯符号/纯表情/无语义文本，判 `anger=0`。
5. 无法判断时在 `notes` 记录原因，提交仲裁（`adjudicated_label`）。

## 4. 示例（至少 20 条）

### 4.1 正例（anger=1）

1. “真是气死我了，这种事还能发生？”
2. “你们这是欺负人吧，太过分了！”
3. “看得我火大，相关方必须出来道歉。”
4. “这种操作真恶心，毫无底线。”
5. “别洗了，就是你们的问题，赶紧负责！”
6. “每次都这样敷衍，真想骂人。”
7. “离谱到家了，完全不能忍。”
8. “我真的被激怒了，这种规定有病吧。”

### 4.2 反例（anger=0）

9. “我觉得这个功能不太好用。”
10. “有点失望，希望下次改进。”
11. “今天心情很低落。”
12. “看到这个新闻有点难过。”
13. “[叹气]”
14. “。。。。。。”
15. “哈哈哈哈”
16. “有点害怕，不知道怎么办。”

### 4.3 边界例（需记录 notes 并优先仲裁）

17. “真行啊你们，又来这一套。”（可能是讽刺，也可能是怒）
18. “我服了。”（语义依赖上下文，可能是无奈）
19. “这操作真绝。”（可能夸赞也可能贬损）
20. “谢谢你们，问题一个都没解决。”（反讽，需看怒气强度）
21. “无语了。”（常见边界词，可能无奈也可能愤怒）
22. “真不知道该说什么。”（情绪方向不确定）

## 5. “道德愤怒”扩展接口（本轮预留）

后续扩展 `moral_anger` 时，可在当前规则上增加以下必要条件：
- 涉及道德/规范违背（如“不公”“失德”“伤害弱者”）。
- 出现责任归因（点名行为主体、机构或群体）。
- 包含道德谴责或应然判断（“应该被追责”“不该这样”）。

建议后续标签体系：
- `anger`
- `moral_anger`
- `other_negative`
- `neutral_or_other`

建议实现方式：
- 第一层：`anger` 二分类（本轮）。
- 第二层：在 `anger=1` 子集上细分 `moral_anger` 与其他类别，或直接端到端多分类。
